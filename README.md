# ⚽ Kreisliga Manager

Ein minimalistisches Fußball-Manager-Spiel im Stil der 90er Jahre.

## 🧠 Idee

Der Spieler übernimmt ein Kreisliga-Team und trifft einfache Entscheidungen.
Die Saison wird simuliert und kann mit der realen Entwicklung verglichen werden.

## 🚀 Features (aktuell)

* einfache Liga mit Teams
* Spieltag-Simulation
* Tabelle mit Punkten und Toren
* Fortschritt wird im Browser gespeichert

## 🔜 Geplant

* echter Spielplan (Saison-System)
* Team-Auswahl
* Taktik-System
* echte Kreisliga-Daten

## 🛠️ Technik

* HTML
* CSS
* JavaScript (Vanilla)
* Speicherung über localStorage

## ▶️ Starten

Einfach `index.html` im Browser öffnen.
